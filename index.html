Exam Grades Dashboard

Analyze the grades of 85 students emphasizing 3 major exams in a dynamic dashboard.

January 09, 2022


This dashboard utilizes parameters and calculated fields to adjust the data in each plot according to user selection of a particular exam (Exam #1, Exam #2, or Exam #3). In addition, plots can be filtered by class period and student demographic data. Various .csv documents from multiple data sources were converted to a relational database and joined using SQL in pgAdmin. The final export table was cleaned and prepped in excel before importing to Tableau.

Personal Finance Dashboard

A categorical time series of net worth visualizing the magnitude of various assets and liabilities.

December 28, 2021

A relational database consisting of nine financial accounts (e.g. checking accounts, savings, loans) was created and queried using pgAdmin to create a Tableau dashboard that tracks income, spending, and net worth. Some metrics and axes in the dashboard are hidden for general privacy.
